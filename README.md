
# Smart Energy Agentic AI

## Overview

An Agentic AI based Smart Home Energy Management System using:

- A2A (Agent-to-Agent Communication)
- MCP (Model Context Protocol Inspired Tool Layer)
- RAG (Retrieval Augmented Generation)
- Open Source LLMs

## Architecture

User
↓
Energy Orchestrator Agent
↓
Forecast Agent
Optimization Agent
Cost Analysis Agent
Reporting Agent
Security Agent
↓
A2A Communication Layer
↓
MCP Tool Layer
↓
RAG Knowledge Base

## Features

- Multi-Agent Collaboration
- Smart Energy Optimization
- Cost Analysis
- Weather Forecast Integration
- Battery Management
- Security Validation
- Knowledge Retrieval

- Python
- Streamlit
- ChromaDB
